#EventFinder

## Features
- **RecyclerView Implementation**: Displays a list of events in a structured and scrollable layout.
- **Filter and Sort Options**: Allows users to filter events based on name, city, or date.
- **Detail Screen**: Presents event details in a user-friendly layout with data binding enabled for efficient data display.
- **Custom Styling**: The app includes unique design elements, including themed colors and layout styling.
  
## Technologies Used
- **Kotlin**: The primary language used for app development.
- **Android Studio**: IDE for building and testing the application.
- **Data Binding**: Enabled for data display on detail screens.
- **RecyclerView**: Used for efficient scrolling and displaying a large list of events.

## How to Run
1. Clone the repository:
    ```
    git clone https://github.com/AlpKucuk1/Eventfinder
    ```
2. Open the project in **Android Studio**.
3. Build and run the application on an **Android Emulator** or physical device.

## Directory Structure
- **app/src/main/java/com.example.assignment1**: Contains the main Kotlin source code files.
- **app/src/main/res/layout**: Contains the XML layout files for the UI components.
- **app/src/main/assets/events**: Holds CSV data files for loading event data.
